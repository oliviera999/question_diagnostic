✨ QUOI DE NEUF ?
═══════════════════════════════════════════════════════════════════

Vous pouvez maintenant accéder DIRECTEMENT à n'importe quelle 
catégorie dans la banque de questions native de Moodle !


🎯 COMMENT ÇA MARCHE ?
═══════════════════════════════════════════════════════════════════

2 façons d'accéder à une catégorie :

  1️⃣  CLIQUER SUR LE NOM
      ├─ Dans le tableau
      ├─ Nom de la catégorie = lien bleu 🔗
      └─ S'ouvre dans un nouvel onglet

  2️⃣  BOUTON "👁️ VOIR"
      ├─ Colonne "Actions"
      ├─ Bouton bleu avec icône œil
      └─ S'ouvre dans un nouvel onglet


📊 EXEMPLE VISUEL
═══════════════════════════════════════════════════════════════════

AVANT :
  ┌──────────────────────────────────────────────┐
  │ Nom            │ Actions                     │
  ├────────────────┼─────────────────────────────┤
  │ Mathématiques  │ [🗑️ Supprimer] [🔀 Fusionner] │
  └────────────────┴─────────────────────────────┘
  
  → Copier ID → Naviguer manuellement → Chercher
  ⏱️ Temps : ~2 minutes


APRÈS :
  ┌──────────────────────────────────────────────────────────┐
  │ Nom                    │ Actions                         │
  ├────────────────────────┼─────────────────────────────────┤
  │ 🔗 Mathématiques       │ [👁️ Voir] [🗑️ Sup.] [🔀 Fus.]  │
  │    ↑ Cliquez ici !     │      ↑ Ou ici !                 │
  └────────────────────────┴─────────────────────────────────┘
  
  → 1 clic → Ouverture automatique
  ⏱️ Temps : ~5 secondes

  💰 GAIN : 95% de temps en moins !


💡 CAS D'USAGE
═══════════════════════════════════════════════════════════════════

🔍 VÉRIFICATION RAPIDE
   1. Voir une catégorie suspecte dans l'outil
   2. Clic sur le nom → Vérification dans la banque
   3. Retour à l'outil (onglet toujours ouvert)
   4. Décision : supprimer ou conserver

🧹 NETTOYAGE ASSISTÉ
   1. Filtrer "Vides"
   2. Pour chaque catégorie : Clic "👁️ Voir"
   3. Vérifier qu'elle est vraiment vide
   4. Retour et suppression si OK

🔀 FUSION INTELLIGENTE
   1. Clic "👁️ Voir" catégorie A → Onglet 2
   2. Clic "👁️ Voir" catégorie B → Onglet 3
   3. Comparer le contenu
   4. Décider laquelle garder
   5. Fusionner en connaissance de cause

📋 WORKFLOW MULTI-ONGLETS
   Onglet 1 : Outil de diagnostic (vue d'ensemble)
   Onglet 2 : Catégorie A (détails)
   Onglet 3 : Catégorie B (détails)
   Onglet 4 : Catégorie C (détails)
   
   → Travail parallèle
   → Comparaison facile
   → Contexte préservé


🎨 DESIGN
═══════════════════════════════════════════════════════════════════

✅ Nom de catégorie cliquable (bleu Moodle)
✅ Icône 🔗 discrète (opacité 50%)
✅ Bouton "👁️ Voir" bien visible (bleu primaire)
✅ Hover avec soulignement
✅ Ouverture dans nouvel onglet (target="_blank")
✅ Style cohérent avec le reste de l'interface


⚠️ NOTES IMPORTANTES
═══════════════════════════════════════════════════════════════════

• Les catégories ORPHELINES (contexte invalide) n'ont pas de lien
  → Normal : leur contexte n'existe plus dans Moodle
  → Le nom s'affiche en texte simple (non cliquable)

• Permissions requises :
  → Vous devez avoir accès au cours/contexte de la catégorie
  → Sinon, erreur 404 ou "Access denied"

• Compatible Moodle 4.3+ :
  → Testé sur Moodle 4.5
  → Format URL standard utilisé

🚀 INSTALLATION DE LA MISE À JOUR
═══════════════════════════════════════════════════════════════════

Si vous utilisez déjà la version 1.0.0 :

  1. Remplacer les fichiers suivants :
     ✅ classes/category_manager.php
     ✅ index.php
     ✅ styles/main.css
     
  2. Purger les caches Moodle :
     Administration > Développement > Purger les caches
     
  3. Rafraîchir la page (Ctrl+F5)
     
  4. ✅ C'est prêt !
